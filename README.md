## 效果示例

```
用户      ❯ AME，我对线总是被打爆怎么办？感觉对面中单比我强，我很烦。

AME      ❯ 对线被打爆嘛，额，其实无所谓啊。

            你觉得对面比你强，那说明他确实打得挺好的。
            但你觉得烦有什么用呢？烦又不能让你变强呀。

            我打得好我们就能赢，我打得差就会输——
            所以你就focus在自己身上嘛。

            别去想对面多屌，你只需要想一件事：
            我这波兵怎么处理，这个补刀怎么不漏。

            对线打不过，中期刷起来不就行了。
            一号位又不靠对线赢比赛，知道吧？
```

```
用户      ❯ 820，我们队优势局总是被翻盘，怎么稳住？

820      ❯ 伟乌生，优不盾——优势的时候不要去打盾。

            兄弟们，你们这个毛病我太熟了。
            优势局被翻盘，说白了就一个字：贪。

            优势局的核心不是"怎么赢更快"，是"怎么不犯错"。
            你把兵线控好，视野做足，
            对面出来就接，不出来就耗。

            优势在你这边，时间也在你这边，你急什么？

            我这人脑子不活络，但感觉还在。
            打多了你就知道，那种"这波该不该打"的直觉，
            比什么战术分析都管用。
```

```
用户      ❯ Topson，我们队中单和carry总是抢资源打架怎么办？

Topson   ❯ 我就说一句——几乎所有中单选手都太自私了。

            他们只顾着自己肥，并不了解DOTA的真正含义。
            这是团队合作的游戏，无论你扮演什么角色，
            都必须考虑什么样的打法最适应团队节奏。

            你中单安心发育没错，但一个优秀的中单
            还需要去做很多打钱以外的事情。

            这种博弈才是DOTA2最吸引人的地方。

            你们需要坐下来谈清楚：
            谁在什么时间点是最强的，
            资源应该向谁倾斜才能一起赢。

            不是你肥就能赢，是你们一起赢才叫赢。
```

这不是角色扮演。他们用的是「即时理解 > 经验」、「团队DOTA > 个人能力」、「优势不贪」这些真实的认知框架在帮你分析——**不是复读语录，是用他们的思维方式帮你看问题。**

## 诚实边界

**这个Skill能做的：**
- 用六位DOTA传奇的思维框架分析游戏问题（对线、团战、BP、心态等）
- 模拟各自的语言风格和表达习惯
- 提供他们在竞技DOTA领域的视角和智慧

**做不到的：**

| 维度 | 说明 |
|------|------|
| 商业决策 | 除伍声2009外，其他人的商业思维数据不足 |
| 实时版本 | 素材截止2025-2026年，最新版本变化可能有缺漏 |
| 替代本人 | 他们的当下状态、最新想法无法被复制 |
| Wings内部 | 与俱乐部纠纷等敏感话题，素材不足以准确呈现 |
| 英雄教学 | 他们讲的是思维框架，不是操作教程 |

**一个不告诉你局限在哪的Skill，不值得信任。**
